## 🌟 项目简介

**12306全国查票系统** 是一个基于Python和Tkinter开发的桌面应用程序，旨在为用户提供更便捷、更强大的火车票查询体验。本项目通过调用12306官方公开接口，实现了**9大核心功能**，覆盖了从简单查票到智能换乘的全场景需求。

无论你是需要查询某条线路的实时余票，还是想规划复杂的换乘路线，亦或是监控心仪车次的余票变化，这款工具都能帮你轻松搞定！

## ✨ 核心功能一览

| 模块 | 功能描述 | 状态 |
|------|----------|------|
| 🔍 **自主查询** | 按出发站、到达站、日期查询车次，支持车型、时间段、价格范围筛选，多种排序方式 | ✅ 已完成 |
| 🌍 **全国查询** | 遍历全国主要城市之间的所有车次，进度条显示，支持暂停/继续/清空 | ✅ 已完成 |
| 🚅 **车次查询** | 输入车次号查询完整时刻表，包含所有经停站信息，支持热门车次快捷按钮 | ✅ 已完成 |
| 🏢 **车站大屏** | 模拟车站实时大屏，区分出发/到达列车，状态颜色标记（已发车/计划中/已到达/停运） | ✅ 已完成 |
| 📊 **票价趋势** | 查询连续7天的票价变化，对比各席别价格 | ✅ 已完成 |
| 🚉 **车站车次** | 查询指定车站的所有经过车次，区分出发和到达列车 | ✅ 已完成 |
| 📈 **正晚点统计** | 查询列车历史正晚点记录（目前为模拟数据） | ⚠️ 暂未完全实现 |
| 🔔 **余票提醒** | 添加车次余票监控，定时查询，有票时弹窗提醒 | ✅ 已完成 |
| 🔄 **智能换乘** | 基于BFS算法的全国换乘搜索，支持同省份优先策略、等候时间过滤、12/24小时制切换 | ✅ 已完成 |

## ⚠️ 重要说明

### 🔸 功能状态说明
**目前部分功能暂未完全实现，请谅解！** 🙏

- `正晚点统计` 模块目前仅展示**模拟数据**，因12306官方未提供历史正晚点接口，后续计划接入第三方数据源或通过爬虫实现。
- `车站大屏` 的实时状态是基于当前时间模拟的，并非真正的实时数据，但余票和时刻表均为官方真实数据。
- `智能换乘` 算法仍在优化中，部分偏远地区的换乘方案可能不够完善。

### 🔸 数据准确性声明
**我们无法保证数据完全正确，请以12306官方实际数据为准！** ⚠️

本程序所有数据均来自中国铁路12306官方公开接口，但由于网络延迟、接口变更、数据缓存等原因，可能会出现以下情况：
- 余票信息与实际略有差异
- 时刻表与官方最新调整不符
- 部分车次信息缺失

**建议在购票前，务必登录12306官网或官方APP核实最终信息！**
