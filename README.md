## ✨ 项目简介

**12306全国查票系统** 是一款功能强大的桌面端火车票查询工具，经过多轮迭代优化，现已推出**最终版 v3.0**。本版本**榨干了所有可用接口**，整合了12306官方数据与12036第三方实时信息，实现了**9大核心功能**，覆盖从查票、换乘到实时监控的全场景需求。

无论你是需要查询某条线路的余票，还是想规划最优换乘路线，亦或是实时关注车站大屏状态，这款工具都能帮你轻松搞定！

---

## 🚀 核心功能

| 模块 | 功能描述 | 数据源 |
|:-----|:---------|:-------|
| 🔍 **自主查询** | 按出发站、到达站、日期查询车次，支持车型筛选、时间段筛选、价格范围筛选，多种排序方式 | 12306官方 |
| 🌍 **全国查询** | 遍历全国主要城市之间的所有车次，进度条显示，支持暂停/继续/清空 | 12306官方 |
| 🚅 **车次查询** | 输入车次号查询完整时刻表，包含所有经停站信息，支持热门车次快捷按钮 | 12306官方 + 12036第三方 |
| 🏢 **车站大屏** | **实时车站信息**，包含车次、始发站、终点站、发车时间、检票口、状态、晚点。颜色标记：黑色(默认)/绿色(检票中)/红色(停止检票/停运) | 12036第三方 |
| 📊 **票价趋势** | 查询连续7天的票价变化，对比各席别价格（二等座、一等座、商务座、软卧、硬卧） | 12306官方 |
| 🚉 **车站车次** | 查询指定车站的所有经过车次，区分出发和到达列车，显示完整运行区间 | 12306官方 |
| 📈 **正晚点统计** | 实时晚点查询（从车站大屏接口获取） | 12036第三方 |
| 🔔 **余票提醒** | 多车次监控，定时查询，有票弹窗提醒，支持多车次同时监控 | 12306官方 |
| 🔄 **智能换乘** | 官方换乘接口，包含总历时、总票价、等候时间 | 12306官方 |
