# 🚄 12306 全国查票系统

## 📥 下载使用
- `全国查票系统V3.0.exe` - 直接双击运行（无需安装Python）
- `源代码.py` - 源代码（需要Python环境）

## ✨ 功能列表
- 🔍 自主查询
- 🌍 全国查询
- 🚅 车次查询
- 🏢 车站大屏
- 📊 票价趋势
- 🚉 车站车次
- 📈 正晚点统计
- 🔔 余票提醒
- 🔄 智能换乘

## 🔗 数据来源

本程序所有数据均来自**中国铁路12306官方接口**，包括但不限于：

| 接口 | 功能 |
|------|------|
| `/otn/leftTicketPrice/queryAllPublicPrice` | 票价+余票 |
| `/otn/leftTicket/query` | 余票查询 |
| `/otn/queryTrainInfo/query` | 时刻表查询 |
| `/otn/queryTrainInfo/queryTrainDelay` | 正晚点查询 |
